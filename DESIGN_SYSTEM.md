## Responsive Design

### Portrait (Mobile)

- Width: Full screen with 16px padding
- Card height: Auto based on content
- Charts: Full width, 300px height
- Font sizes: As specified

### Landscape (Tablet)

- 2-column grid for stats
- Wider charts
- Adjusted padding
- Larger typography

## Accessibility Features

### Text Contrast

- All text meets WCAG AA standards
- Dark text on light backgrounds
- Light text on dark backgrounds

### Touch Targets

- Minimum 48x48 dp for interactive elements
- Sufficient spacing between buttons
- Large text fields for input

### Icons

- Always paired with text labels
- Meaningful symbols
- Consistent across app
